FAQ

Is it really fair?

Yes. We use public randomness from drand and derive a seed from your dataset hash and the drand value. The Fisher–Yates shuffle ensures uniform selection.

Do you store my data?

No data is stored by default when you run locally. You can keep the receipt JSON if you want an audit trail.

Can I reproduce a draw?

Yes. Using the same dataset, settings, and drand round from the receipt, you can reproduce the winners.
